## Problem

Create a P5 sketch that loads an MP3 file. Use the following file path to access the MP3:

```tex
'music/Beat-Culture-Midori.mp3'
```

Then, create a button with the label `Play/Stop`. This button should play the MP3 if it is not playing, and then stop it if it is playing.

### Running the Sketch

The panel in the bottom-left will display your sketch. If you make changes to the JavaScript file, you need to reload the bottom-left panel. Click on the icon with two blue arrows in a circular shape. This will reload the webpage with our JavaScript output.

![reload icon](.guides/img/refresh.png)

|||info
## Loading the MP3
It can take sometime to load the MP3. You should see the text `"Loading..."` appear in the bottom left panel. After a few moments, your sketch should be ready to go.

|||
