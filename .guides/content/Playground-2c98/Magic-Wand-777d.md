
## Content Generation

The Magic Wand is Codio's tool to generate content with the help of AI.

![.guides/img/magic-wand-tool](.guides/img/magic-wand-tool.png)

Provide the tool with a prompt, and click "GENERATE".

```text
Create a lesson on how to use recursion to find the nth digit of the Fibbonacci sequence. Be sure to show students how memoization can improve performance.
```

Codio uses the human-in-the-loop approach with the magic wand. Content authors must approve the changes before they appear on the page. Using the content preview, you can iteratively improve the content by adjusting the prompt.

Codio provides a system prompt for the magic wand, such that you only need to specify the topic.

This _is not_ a "one-click-and-go" solution. You will need to adapt the page to the lesson to make it interactive.
