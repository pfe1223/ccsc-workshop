
## Quickly Generate Formative Assessments

We saw how the AI-generated assessments automatically use the current page as context for your question. You should not have to spend extra time crafting a prompt when so much vital information is already on the page.

### Adding Variation to Assessments

If you do not provide a prompt, the AI assessment tool often resorts to a more "summary" style question based on the page of content. This may not always be what you want students to answer. Here are some prompt variations that can push your students beyond reading comprehension of the page.

* Have students choose between multiple correct answers.
  ```text
  Have multiple correct answers.
  ```
* Give students a faulty code snippet, and ask them to identify the error.
  ```text
  Give students a faulty code snippet, and have them identify the error.
  ```
* Give students a code snippet and ask students to predict the output.
  ```text
  Give students a code snippet and have them predict the output.
  ```

### Fill in the Blanks

We can also create fill in the blank questions with the help of an AI-based tool.

```text
Give students definitions for recursion, base case, dynamic programming, and memoization. Have them select the correct term.
```

### Parsons Problems

This type of assessment has students select among several code fragments. They must construct the correct solution from the given fragments.

```text
Have students select the correct Python code that creates a recursive function that calculates the length of a list.
```
