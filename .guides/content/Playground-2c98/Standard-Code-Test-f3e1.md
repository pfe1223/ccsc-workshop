
## Input/Output Testing

Standard code test is the name Codio uses for input/output testing. These types of assessments can be built with the help of an AI assistant.

After generating the assessment, scroll down to the bottom of the preview. Here, you will see the name of the created file (containing the solution), as well as the command needed to run the student code. The file should also appear in the file tree.

```text
Have students create a recursive function that calculates the factorial for a given integer.
```
